# Installation
The core packages of the gospecfunc suite are written in pure Go. Installation is done using go get.
```
go get -u github.com/dreading/gospecfunc
```

# Testing 
```
 cd erf
 go test 
```
# Benchmarking
```
 cd erf
 go test -bench=.
```
